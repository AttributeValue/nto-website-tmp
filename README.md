## üöÄ Deploying the Website

You can easily deploy this site to **GitHub Pages** or host it elsewhere if you already have web hosting experience.

---

### Option 1: Deploy via GitHub Pages (Recommended)

1. Go to your repository on **GitHub**.  
2. Click **Settings** ‚Üí **Pages** (under **Code and automation** in the sidebar).  
3. In the **Build and deployment** section:
   - Under **Source**, select **Deploy from a branch**.
   - Under **Branch**, choose `main`, then click **Save**.  
4. After a few moments, your site will be live at: `https://<your-username>.github.io/nto-website/` (replace `<your-username>` with your GitHub username).

---

### Option 2: Fork the Repository (for first-time users)

If you don‚Äôt already have a GitHub account or repository fork:

5. Then follow the deployment steps above to publish via GitHub Pages.
