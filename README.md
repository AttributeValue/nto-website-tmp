# Northern Trail Outfitters Website

**Northern Trail Outfitters (NTO)** is a demo B2C eCommerce website built entirely with client-side scripts.  
It emulates a realistic online shopping experience, making it ideal for learning, demos, or showcasing front-end capabilities without requiring a back-end server.

## Deploying the Website

You can easily deploy this site to **GitHub Pages** or host it elsewhere if you already have web hosting experience.

1. If you donâ€™t already have a GitHub account, visit [GitHub](https://github.com) and create an account.  
1. Once logged in, [click here to fork this repository](https://github.com/eliotharper/nto-website/fork).  
1. In the **Owner** dropdown, select your account.  
1. Click **Create fork** to copy this repository to your account.
1. Click the **Settings** tab.
1. Select **Pages** (under **Code and automation** in the sidebar).  
3. In the **Build and deployment** section, under **Source**, select **Deploy from a branch**.
1. Under **Branch**, choose `main`, then click **Save**.  
1. After a few moments, your site will be live at: `https://<your-username>.github.io/nto-website/` (replace `<your-username>` with your GitHub username).
